TPV GB - Sistema de Punto de Venta
===================================

Repositorio GitHub con código fuente completo:
https://github.com/gnnesis/TPV-DesarrolloWeb

CONTENIDO DEL REPOSITORIO:
- presentation/TPV_GB.pptx (Presentación del proyecto)
- src/ (Código fuente de los 4 microservicios)
- docker-compose.yml (Orquestación con Docker)
- README.md (Documentación completa)

EJECUCIÓN RÁPIDA:
1. Instalar Docker Desktop
2. Clonar el repositorio
3. Ejecutar: docker-compose up
4. Acceder a: http://localhost:8080

Para instrucciones detalladas, consultar el README.md del repositorio.
